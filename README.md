# 🎮 Grid-Sweeper (网格扫雷)

A minimalist, pure frontend implementation of the classic Minesweeper game, featuring a sleek Dark Mode & Glassmorphism aesthetic. Built as a geeky easter egg for data infrastructure and tech enthusiasts.

## ✨ Features
- **Pure Frontend**: HTML5, CSS3, and Vanilla JavaScript. Zero dependencies.
- **Geek Aesthetic**: Dark gradient backgrounds and glass-like UI components.
- **Smart Mechanics**: Prevents immediate loss on the first click and implements efficient recursive blank space revealing.
- **Custom Modals**: Clean, animated pop-ups instead of native browser alerts.
- **Difficulty Levels**:
  - Beginner (9x9, 10 mines)
  - Intermediate (16x16, 40 mines)
  - Expert (30x16, 99 mines)

## 🚀 How to Play
Simply clone the repository and open `index.html` in your favorite web browser.

```bash
git clone https://github.com/knight008848/Grid-Sweeper.git
cd Grid-Sweeper
# Open index.html
```
